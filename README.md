# Sistema de Gestión de Pagos de Agua - Java

Aplicación de escritorio desarrollada en Java para la gestión de usuarios y registro de pagos de agua, orientada a resolver necesidades de un comité local.

## Descripción

Sistema diseñado para administrar información de usuarios y sus pagos, implementando una interfaz gráfica y lógica de negocio estructurada.

## Funcionalidades

- Registro de usuarios
- Registro de pagos
- Visualización de datos en tablas
- Actualización de información
- Interfaz gráfica con Java Swing

## Arquitectura

El proyecto sigue una estructura organizada por capas:

- **model** → Entidades (Usuario, Pago)
- **service** → Lógica de negocio
- **view** → Interfaz gráfica
- **main** → Punto de entrada

## Tecnologías

- Java
- Java Swing
- Programación orientada a objetos
- Arquitectura por capas (MVC)

## Objetivo

Desarrollar un sistema funcional para la gestión de pagos de agua en un entorno real.
